# 🎓 Graduation Gift Website

Website hadiah wisuda yang interaktif dan responsif.

## 📁 Struktur Folder

```
├── index.html          ← File utama website
├── style.css           ← Semua styling
├── script.js           ← Animasi & interaksi
├── photos/             ← Taruh foto-foto di sini
│   ├── photo1.jpg      ← Foto polaroid 1
│   ├── photo2.jpg      ← Foto polaroid 2
│   ├── photo3.jpg      ← Foto polaroid 3
│   ├── photo4.jpg      ← Foto polaroid 4
│   ├── photo5.jpg      ← Foto polaroid 5
│   └── main-photo.jpg  ← Foto utama di section pesan
├── videos/             ← Taruh video di sini
│   ├── video1.mp4
│   ├── video2.mp4
│   └── video3.mp4
└── music/              ← Taruh musik background di sini
    └── background.mp3
```

## ✏️ Cara Kustomisasi

### 1. Ganti Nama
Di `index.html`, cari semua teks `[ Nama Kamu ]` dan ganti dengan nama yang dituju.

### 2. Ganti Foto
Taruh foto-foto ke folder `photos/` dengan nama:
- `photo1.jpg` s/d `photo5.jpg` → untuk grid polaroid
- `main-photo.jpg` → untuk foto di section pesan

### 3. Ganti Video
Taruh video ke folder `videos/` dengan nama:
- `video1.mp4`, `video2.mp4`, `video3.mp4`

### 4. Ganti Musik
Taruh file musik ke folder `music/` dengan nama `background.mp3`

### 5. Ganti Teks
Edit teks di `index.html`:
- Caption polaroid (`.polaroid-caption`)
- Pesan di section message
- Caption video (`.video-caption`)
- Teks closing

## 🚀 Cara Buka

Buka `index.html` langsung di browser, atau jalankan via server lokal (XAMPP, Live Server, dll).
